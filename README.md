# RL Debt Collection Repository

The simulator is located in the **rl-debt-collection** directory. The examples that use the simulator are located in the **examples** directory. The simulator was created as a **uv** library project. The examples are **uv** binary projects. Follow [instructions](https://docs.astral.sh/uv/getting-started/installation/) to install **uv**.

## Installation

To download the repository, correctly synchronize your git installation with github account and execute in the desired directory:

```bash
git clone https://github.com/SKN-Business-Analytics/rl-debt-collection.git
cd rl-debt-collection
```

## Run example

To run the basic blank example execute:

```bash
cd examples/blank
uv run main.py
```
